# Data Analysis & Visualization
This directory contains the Jupyter notebooks and Python modules required to process the raw statistical data (.dat files) generated by the benchmarking pipeline. It provides a structured workflow to aggregate results, analyze estimator performance, and generate publication-ready tables and graphs.

## Analysis Notebooks
The Jupyter notebooks represent the actual data processing and visualization pipeline.

### General Benchmark Outputs
- `generate_big_graph.ipynb`: Generate the main comparative graphs. Produce similar graphs as the ones in the paper.

- `generate_big_tabs.ipynb`: Aggregates the error metrics into massive summary tables. It is designed to output clean, formatted tables.

- `convergence.ipynb`: Analyzes the convergence of the different estimators. 

- `iteration_mls.ipynb`: A targeted analysis investigating the impact of Moving Least Squares (MLS) iterations.

### Dataframes to web
- `createLightDataframes.ipynb`: This file extract the resulted .dat files to create specific .tsx dataframes, make it works and readable in our website.
